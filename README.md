# Gemini TeamSync 🤝

A **Flask-based web application** that acts as a unified team hub powered by Google Gemini AI. Designed to keep teams in sync with AI-assisted collaboration.

## ✨ Features

- 🤖 **Google Gemini AI** integration for intelligent team assistance
- 🌐 **Flask web server** with HTML templates
- 👥 **Team Hub** — centralized collaboration interface
- ⚡ Lightweight and easy to deploy

## 🚀 Getting Started

### Prerequisites
- Python 3.8+
- Google Gemini API key

### Installation

```bash
# Clone the repo
git clone https://github.com/Eshwarkadari/Gemini-TeamSync.git
cd Gemini-TeamSync

# Install dependencies
pip install -r requirements.txt

# Run the app
python app.py
```

Open **http://localhost:5000** in your browser.

## 🔧 How It Works

1. Flask serves the web interface
2. Team members interact through the hub
3. Gemini AI assists with team tasks and queries
4. All responses rendered via HTML templates
